# Trading Sphere – Awealth

A comprehensive, multi-market trading calculator designed for traders and developers.  
Supports Crypto, Forex, Stocks, and Commodities. Calculates risk, position size, margin, leverage, and risk/reward ratio.  
Features:

- Automatic LONG / SHORT detection  
- Multi-currency support (USD, USDT, BTC, ETH)  
- Risk % and account balance-based calculations  
- Leverage and margin-aware position sizing  
- Trade history tracking  
- Built with HTML, CSS, JavaScript (fully offline-capable)  

This project is perfect for traders, students, and developers who want a reliable, offline-friendly tool to manage trading calculations efficiently.
